# TP Integrador - Panel de Control de Clientes

## Descripcion breve del proyecto
Aplicacion web desarrollada con React + Vite para gestionar clientes de una empresa. Incluye autenticacion de administrador con Context API y LocalStorage, listado de clientes con consumo de API (FakeStoreAPI), buscador dinamico, alta de clientes con formulario (POST), vista de detalle por cliente y control de permisos por sector para acciones de eliminacion (DELETE simulado).
